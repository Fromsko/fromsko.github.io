# 开发笔记

本仓库包含按技术和主题组织的各种开发笔记。以下是每个子目录的结构和内容概述。

## 目录结构

- **android/**: 包含与 Android 开发相关的笔记。

  - **harmony/**: HarmonyOS 开发相关笔记。
    - `Base.md`: HarmonyOS 的基本概念和设置。
  - **imgui/**: 在 Android 中使用 ImGui 的相关笔记。
